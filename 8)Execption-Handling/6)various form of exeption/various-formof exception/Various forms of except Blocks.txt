			============================================================
					Various forms of except Blocks
			============================================================
---------------------------------------------------------------------------------------------------------------------------------------------------------
Form-1 : This except block can handle one exception at a time
---------------------------------------------------------------------------------------------------------------------------------------------------------
	try:
	     ---------------------------------------------
	     ---------------------------------------------
	except <exception-class-name-1>:
	       --------------------------------------------
	       --------------------------------------------
	except <exception-class-name-2>:
	       --------------------------------------------
	       --------------------------------------------
	---------------------------------------------------
	except <exception-class-name-n>:
	       --------------------------------------------
	       --------------------------------------------
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Form-2 : This except block can handle Multiple Specific Exception at time--Multi Exception Handling Block.
------------------------------------------------------------------------------------------------------------------------------------------------------------------

		try:
			-------------------------------------------------------------
			Block of statements generates exception
			-------------------------------------------------------------
		except <exception-class-name-1,exception-class-name-2,...exception-class-name-n>:
		      ---------------------------------------------------------------------------
		      Block of statements generaes Multiple User Friendly
		      Error Messages for all specific Exceptions
		      ---------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Form-3 : This except block can handle all types of Un-Known Exceptions
------------------------------------------------------------------------------------------------------------------------------------------------------------------

		try:
			--------------------------------------
			--------------------------------------
			--------------------------------------
		except :
			-------------------------------------
			User-Friendly Error Message
			-------------------------------------
------------------------------------------------------------------------------------------------------------------------------------------------------------------
Form-4 : final syntax of handling exception
------------------------------------------------------------------------------------------------------------------------------------------------------------------

		try:
			--------------------------------------
			--------------------------------------
			--------------------------------------
		except <exception-class-name-1,exception-class-name-2,...exception-class-name-n>:
		      ---------------------------------------------------------------------------
		      Block of statements generaes Multiple User Friendly
		      Error Messages for all specific Exceptions
		      ---------------------------------------------------------------------------
		except :  # default except block
			-------------------------------------
			User-Friendly Error Message
			-------------------------------------
		else:
		      -----------------------------------------------
		      Block of statements

		finally:
