		=================================================================
					Writing the Data to the File
		=================================================================
=>To write the Data to the file, we have Two Pre-Defined Functions present in the object of TextIOWrapper. They are

				1. write()
				2. writelines()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. write()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:      FilePointerObj.write(strdata)
=>This Function is used for writing or Saving any type of Data to the file in the form of str.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
2. writelines()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:     FilePointerObj.writelines( Iterable-object)
=>This Function is used for writing or saving any type Iterable-object data to the file in the form of str.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
NOTE:	write() and writelines() writing the data to file in the form of Value by Value.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
