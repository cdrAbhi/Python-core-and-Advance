		=================================================================
					Reading the Data from the File
		=================================================================
=>To read the data from the file, we have Two Pre-Defined Function present in the object of TextIOWrapper. They are

			1. read()
			2. readlines()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. read()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:     varname=FilePointer.read()
=>This Function is used for Reading Entire Data of the file and placed in LHS Varname and whose type <class, 'str'>
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2. readlines()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:     varname=FilePointer.readlines()
=>This Function is used for Reading Entire Data of the file  in the form of Line by Line and placed in LHS Varname and whose type <class, 'list'>
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
